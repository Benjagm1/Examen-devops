# Libro de clases — Frontend (Angular)

Frontend único del proyecto: **Angular 19**, patrón **Container / Presentational**.

## Desarrollo

1. Microservicios en Docker (`H/Contenedores`) o locales en **8051–8056**.
2. En esta carpeta:

```powershell
npm.cmd install
npm.cmd start
```

Abre **http://127.0.0.1:8050/** (el proxy está en `proxy.conf.json`).
